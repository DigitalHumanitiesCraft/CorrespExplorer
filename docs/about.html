Ueber CorrespExplorer

Was ist CorrespExplorer?

CorrespExplorer ist ein webbasiertes Werkzeug zur Visualisierung und Exploration von Korrespondenz-Metadaten. Es wurde entwickelt, um Briefeditionen und Korrespondenzprojekte interaktiv erfahrbar zu machen.

Die Anwendung verarbeitet Daten im CMIF-Format (Correspondence Metadata Interchange Format) und bietet verschiedene Ansichten: Kartenvisualisierung, Zeitliche Verteilung, Korrespondentenlisten, Themenexploration und Ortsverzeichnis.

Das CMIF-Format

CMIF (Correspondence Metadata Interchange Format) ist ein auf TEI-XML basierendes Format zum Austausch von Korrespondenz-Metadaten. Es wurde von der TEI Correspondence SIG entwickelt und ermoeglicht die Vernetzung verschiedener Briefeditionen.

CMIF enthaelt typischerweise:

  • Absender und Empfaenger (mit Normdaten-Verknuepfung)
  • Datum und Zeitraum
  • Absendeort (mit Geokoordinaten)
  • Sprache des Briefs
  • Themen und Schlagworte
  • Verweise auf die Quelledition

CMIF-Spezifikation auf correspSearch

correspSearch

correspSearch ist ein Webservice der Berlin-Brandenburgischen Akademie der Wissenschaften, der Briefmetadaten aus verschiedenen Editionen aggregiert und durchsuchbar macht.

CorrespExplorer kann CMIF-Daten direkt von correspSearch laden. Die API ermoeglicht Abfragen nach Korrespondenten, Zeitraeumen und anderen Kriterien.

Umgang mit Datenqualitaet

Historische Korrespondenzdaten enthalten oft Unsicherheiten und Luecken. CorrespExplorer erkennt diese automatisch und zeigt sie transparent an.

Visuelle Symbole fuer Datumsangaben

Unvollstaendiges Datum Nur Jahr oder Jahr-Monat bekannt (z.B. "Jun 1900")
Zeitraum Brief wurde zwischen zwei Daten verfasst (notBefore/notAfter)
Unsichere Datierung Die Quelle markiert das Datum als unsicher (cert="low")
Datum unbekannt Kein Datum in den Metadaten vorhanden

Timeline-Visualisierung

In der Timeline-Ansicht werden Balken mit unscharfen Datierungen durch diagonale Schraffur markiert. Die Schraffurhoehe zeigt den Anteil der Briefe mit ungenauem Datum im jeweiligen Jahr. Details erscheinen im Tooltip.

Weitere erkannte Unsicherheiten:

  • Unscharfe Orte: Regionen wie "Steiermark" ohne exakte Koordinaten
  • Unbekannte Personen: Absender oder Empfaenger als [NN] oder "Unbekannt" markiert
  • Fehlende Normdaten: Personen ohne VIAF/GND-Verknuepfung

Die Statistiken in der Seitenleiste zeigen, wie viele Datensaetze von Unsicherheiten betroffen sind.

Prinzip: Unsicherheit wird transparent dargestellt, niemals versteckt oder durch falsche Praezision ueberdeckt.

Technologie

CorrespExplorer ist eine clientseitige Webanwendung ohne Backend-Abhaengigkeiten:

  • Frontend: Vanilla JavaScript (ES6+), HTML5, CSS3
  • Karten: MapLibre GL JS mit OpenStreetMap-Tiles
  • Datenverarbeitung: XML-Parsing im Browser
  • Export: CSV und JSON

Der Quellcode ist auf GitHub verfuegbar und kann frei verwendet werden.

Datenschutz

CorrespExplorer verarbeitet alle Daten lokal im Browser. Hochgeladene Dateien werden nicht auf einem Server gespeichert. Bei URL-Importen werden die Daten direkt von der angegebenen Quelle geladen.
